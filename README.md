# Desafío 2 - Condiciones

[cite_start]Solución al "Desafío 2 - Condiciones" de Desafío Latam[cite: 1, 3]. [cite_start]Este proyecto pone en práctica la manipulación del DOM y el uso de sentencias condicionales (`if`, `else if`, `else`) en JavaScript[cite: 4, 9].

## 📌 Ejercicios

[cite_start]El proyecto resuelve tres problemas independientes[cite: 8]:

1. [cite_start]**Borde de Imagen:** Una imagen que al hacerle click alterna entre tener o no tener un borde rojo de 2 pixeles[cite: 11].
2. [cite_start]**Stickers:** Una página para seleccionar stickers[cite: 15]. [cite_start]Suma las cantidades ingresadas y valida que el total no supere los 10 stickers, mostrando mensajes dinámicos según el resultado[cite: 17, 18].
3. [cite_start]**Password:** Un validador de contraseñas de tres dígitos usando selectores[cite: 36]. [cite_start]Muestra mensajes de éxito específicos si se ingresa "911" o "714", y un mensaje de error para cualquier otra combinación[cite: 36].
