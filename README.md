# Desafío 2 - Condiciones

Solución al "Desafío 2 - Condiciones" de Desafío Latam. Este proyecto pone en práctica la manipulación del DOM y el uso de sentencias condicionales (`if`, `else if`, `else`) en JavaScript.

## 📌 Ejercicios

El proyecto resuelve tres problemas independientes:

1. **Borde de Imagen:** Una imagen que al hacerle click alterna entre tener o no tener un borde rojo de 2 pixeles.
2. **Stickers:** Una página para seleccionar stickers. Suma las cantidades ingresadas y valida que el total no supere los 10 stickers, mostrando mensajes dinámicos según el resultado.
3. **Password:** Un validador de contraseñas de tres dígitos usando selectores. Muestra mensajes de éxito específicos si se ingresa "911" o "714", y un mensaje de error para cualquier otra combinación.
